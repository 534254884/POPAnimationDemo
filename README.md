### ### 项目简介: 
-  项目开发语言Swift
-  Swift4.0语法, 需用Xcode9打开

#### 1. 所用动画技术点
- `UIView.animation`动画
- [Core Animation核心动画 ](http://www.jianshu.com/p/2b2f49399b7e)
- 第三方框架[Facebook的POP动画](http://note.youdao.com/)

#### 2. Demo所展示的动画
- 折叠图片
- 音量震动条
- 活动指示器
- 微博动画
- 倒计时-计时器
- 类似QQ信息条数的粘性动画
- 类似雷达动画
